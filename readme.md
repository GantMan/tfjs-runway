# TFJS Runway

A tool for packaging TensorFlow JS models for the web.

### Installation

`npm i -g tfjs-runway`

### Usage

**tfjs-runway \<model path\> \<inject path\>**

`tfjs-runway ./_nsfwjs_model ./template.js`

Check out the included demo app for further instructions.
